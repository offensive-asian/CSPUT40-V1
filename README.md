# CSPUT40

The Communist Sputnik 40 is a re-make publicized version of the Turbo American Falcon 40 (AFAL40) a turbo freedns link generator. It's sole purpose is to make the link generating process for proxies a lot less painful and time consuming, the max speed of the AFAL40 was set at roughly 250-500 link an hour. However, with the improved model of the Communist Sputnik 40 I am currently making, it is possible to reach speeds over 1,000 links an hour.
